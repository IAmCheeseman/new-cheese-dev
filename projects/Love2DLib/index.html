The Github repo.

A template for Love2D projects, it's mostly a personal thing, so it's not consistently worked on, and it's incomplete. Currently it has the following:

  • Object Management

    I handle all processing and drawing, so I don't have to worry about it in the future.

    Look here.

  • Vector Tables

    I have a function that generates tables and has helpful functions in them. All functions which take an x and y are overwritten

    Look here.

  • Timers

    I have a function which generates a timer, and has functions like `Timer:isOver()`.

    Look here.

  • Colors

    I have my own color tables, and I overwrite old functions to use these.

    Look here.
